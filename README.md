# Weather App - Real-Time Weather Forecasting

A modern, full-stack weather application that provides real-time meteorological data for cities worldwide. Built with Flask backend and featuring a premium UI with contemporary design elements.


## Features

-  **Real-time weather data** for any city globally
-  **Comprehensive metrics**: Temperature, Feels Like, Humidity, Wind Speed, Atmospheric Pressure
-  **Modern UI/UX**: Gradient backgrounds, glass-morphism effects, smooth animations
-  **Fully responsive**: Works seamlessly on desktop, tablet, and mobile devices
-  **Fast & lightweight**: Custom CSS without heavy frameworks
-  **Interactive elements**: Floating weather icons, hover effects, error animations

##  Technologies Used

### Backend
- **Python 3.x**
- **Flask** - Web framework
- **Requests** - HTTP library for API calls

### Frontend
- **HTML5** - Semantic markup
- **CSS3** - Custom styling with animations
- **JavaScript** - Dynamic interactions

### API
- **OpenWeatherMap API** - Real-time weather data

## Design Features

- **Gradient Backgrounds**: Modern purple gradient theme
- **Glass-morphism**: Semi-transparent card with blur effects
- **Animations**: 
  - Fade-in on page load
  - Floating weather icons
  - Shake effect on errors
  - Smooth hover transitions
- **Typography**: Clean, hierarchical text styling
- **Responsive Grid**: Adaptive layout for weather details

##  API Integration

The app uses OpenWeatherMap API to fetch:
- Current temperature
- Feels like temperature
- Weather description and icon
- Humidity percentage
- Wind speed
- Atmospheric pressure
